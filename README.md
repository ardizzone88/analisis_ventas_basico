# 📈 Análisis de Ventas: De los Datos a la Estrategia

## 📖 Resumen de la Historia
¿Vender más unidades siempre significa ganar más? En este proyecto, analicé un dataset de ventas minoristas para descubrir que nuestro producto más popular no era el más rentable. 

Este análisis transforma filas de datos en una **estrategia de empaquetado (bundling)** para maximizar los ingresos.

## 🛠️ Herramientas Utilizadas
- **Python** (Pandas para la limpieza, Matplotlib para el storytelling visual).
- **Google Colab** (Entorno de desarrollo).
- **Dataset:** Simulación de ventas de periféricos y hardware.

## 💡 El Hallazgo (The Insight)
Tras procesar los datos, identifiqué una **paradoja de volumen vs. valor**:
1. El **Mouse** representa el 50% de nuestras unidades vendidas.
2. Sin embargo, las **Laptops** generan el 85% de los ingresos totales.

> **Insight Clave:** El esfuerzo logístico de vender periféricos individuales tiene un retorno menor comparado con las unidades de alto valor.

## 🚀 Recomendación de Negocio
Basado en los datos, propongo dejar de promocionar el Mouse de forma aislada y crear un **"Starter Kit"** que incluya Laptop + Mouse + Audífonos. Esto elevaría el valor percibido y simplificaría la decisión de compra del cliente.
